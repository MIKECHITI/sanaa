# Contributing

Thanks for your interest in contributing to this project.

## How to contribute

1. Fork the repository and create a new branch for your feature or fix.
2. Make your changes in a clear and focused commit.
3. Test your changes locally on XAMPP and verify that the site still works.
4. Submit a pull request with a short description of what changed.

## Code style

- Keep PHP simple and readable.
- Use prepared statements for database access and avoid SQL injection.
- Sanitize output with `htmlspecialchars()` or the existing helper functions.

## Reporting issues

If you find a bug or want to request a feature, open an issue with:

- a short summary
- steps to reproduce
- expected behavior
- actual behavior
